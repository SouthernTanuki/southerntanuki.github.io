# A random af Link in bio page

A Link in bio page 

## Features

- **Easy Customization**: Change colors, fonts, and content in seconds.
- **Zero Dependencies**: No frameworks required.

## Quick Start

1. **Download** the files (`index.html`, `404.html`,  `style.css`, `script.js`, `README.md`).
2. **Create an `assets` folder** in the same directory.
3. **Add your images**:
   - `profile.jpg` (Your profile picture, square recommended)
   - `background.jpg` (Your background image, 1920x1080 recommended)
   - `favicon.ico` (Optional, for browser tab icon)
4. **Edit `index.html`**:
   - Replace "Your Name" and "Your bio..." with your info.
   - Update the `href` links for your social media.
5. **Open `index.html`** in your browser to test.

## Customization

### Changing Colors
Open `style.css` and modify the `:root` variables at the top:

### Changing the Background
In style.css, update the --bg-image variable or the background-image property in the body selector.

### Adding More Links
Copy and paste a `<a href="..." class="social-link">...</a>` block inside the .social-links div in index.html.
